# 模型选择指南

## 🎯 如何选择合适的模型

ComfyUI-UnlimitAI 支持用户自定义选择模型，本指南帮助您根据需求选择最佳配置。

---

## 📊 工作流文件说明

### 1. 配置器工作流

**文件**: `configurator.json`

**用途**: 
- 查看模型对比
- 估算成本
- 选择合适配置

**使用方法**:
```
1. 加载 configurator.json
2. 在「配置中心」选择模型
3. 查看「成本估算」
4. 记录您的选择
```

### 2. 主工作流

**文件**: `novel_to_drama_complete.json`

**用途**: 
- 实际运行小说转漫剧

**使用方法**:
```
1. 加载 novel_to_drama_complete.json
2. 根据配置器的选择，在每个节点选择对应模型
3. 填入 API Key
4. 运行工作流
```

## 🎯 如何在 ComfyUI 中选择

### 步骤 1: 查看对比

```
1. 加载 configurator.json
2. 在「配置中心」浏览选项
3. 查看「成本估算」
4. 参考「模型对比」
```

### 步骤 2: 记录选择

```
我的选择：
- 文本模型: deepseek-chat
- 图像模型: flux-pro
- 视频模型: kling-v2
- 音色: male-qn-jingying
```

### 步骤 3: 配置主工作流

```
1. 加载 novel_to_drama_complete.json
2. 节点1 - 选择 text_model: deepseek-chat
3. 节点2 - 选择 image_model: flux-pro
4. 节点3 - 选择 video_model: kling-v2
5. 节点4 - 选择 voice_id: male-qn-jingying
```

### 步骤 4: 运行

```
1. 填入 API Key（4个节点）
2. 输入小说文本
3. 点击 Queue Prompt
```

## ⚠️ 注意事项

### 1. 模型可用性

所有模型都通过 UnlimitAI API 提供，确保 API Key 有效且有足够余额。

### 2. 成本控制

- 先用少量场景测试（3-5个）
- 确认效果满意后再批量生成
- 定期查看账户余额

### 3. 质量与成本平衡

- 视频质量：Kling v2 ≥ VEO 3.1 > VIDU v2
- 视频成本：VIDU v2 < Kling v2 < VEO 3.1
- 性价比：Kling v2 最高

### 4. 处理时间

| 视频模型 | 5秒视频生成时间 |
|----------|----------------|
| VIDU v2 | 1-2 分钟 |
| Kling v2 | 2-3 分钟 |
| VEO 3.1 | 3-5 分钟 |
