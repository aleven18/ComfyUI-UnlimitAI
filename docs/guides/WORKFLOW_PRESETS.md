## 关键发现

### VEO 3.1 限制

⚠️ **VEO 3.1 不支持图生视频**

这会导致图像和视频不一致。所有配置都使用 Kling v2 (图生视频) 来保证一致性。

### 视频成本占比最高

```
视频成本占比:
```
