## 🎯 新增功能

### 4个角色管理节点

#### 1. CharacterImageLoaderNode 👤

**功能**:
- 加载角色参考图
- 提取角色特征
- 生成角色描述
- 维护角色外观一致性

**关键特性**:
- 支持自定义描述或自动生成
- 性别、年龄段、风格可选
- 自动生成seed用于一致性
- 支持角色标签提取

**输出数据**:
```json
{
  "id": "角色唯一ID",
  "name": "角色名称",
  "description": "角色描述",
  "seed": 1234567890,
  "tags": ["标签1", "标签2"]
}
```

#### 2. VoiceDefinitionNode 🎤

**功能**:
- 定义角色音色
- 支持Minimax和OpenAI TTS
- 设置音色参数
- 生成音色预览

**支持的TTS引擎**:
- **Minimax TTS** (中文最佳)
  - 男声: elite, badao, jingqiang
  - 女声: shaonv, yujie, chengshu
  
- **OpenAI TTS** (多语言)
  - 男声: onyx, echo
  - 女声: nova, shimmer

**可调参数**:
- 语速 (0.5-2.0)
- 音调 (0.5-2.0)
- 音色风格
- 情感标签支持

#### 3. CharacterManagerNode 📋

**功能**:
- 注册角色（外观+音色）
- 更新角色数据
- 查询角色信息
- 列出所有角色

**支持的操作**:
- `register` - 注册新角色
- `update` - 更新角色
- `get` - 查询角色
- `list` - 列出角色

#### 4. CharacterConsistencyNode 🎯

**功能**:
- 在图像生成时应用角色特征
- 在视频生成时应用角色特征
- 在音频生成时应用角色特征

**支持的目标类型**:
- `image` - 图像一致性
- `video` - 视频一致性
- `audio` - 音频一致性

## 🔄 使用流程

### 标准工作流

```
步骤1: 注册角色
  ├─ 加载角色图 → CharacterImageLoaderNode
  ├─ 定义音色 → VoiceDefinitionNode
  └─ 注册角色 → CharacterManagerNode

步骤2: 生成场景
  ├─ 获取角色 → CharacterManagerNode(get)
  ├─ 应用一致性 → CharacterConsistencyNode
  ├─ 生成图像 → 图像生成节点
  ├─ 生成视频 → 视频生成节点
  └─ 生成音频 → 音频生成节点

步骤3: 批量处理
  └─ 对所有场景重复步骤2
```

### 示例：创建林晓薇角色

```
1. 加载角色图
   CharacterImageLoaderNode(
     image_url="https://...",
     character_name="林晓薇",
     gender="female",
     age_range="young_adult",
     style="realistic"
   )
   → character_data

2. 定义音色
   VoiceDefinitionNode(
     character_name="林晓薇",
     tts_engine="minimax",
     voice_type="female",
     voice_style="gentle",
     speech_rate=1.0,
     pitch=1.0
   )
   → voice_data

3. 注册角色
   CharacterManagerNode(
     operation="register",
     character_data=character_data,
     voice_data=voice_data
   )
   → character_profile (包含外观和音色)

4. 应用一致性
   CharacterConsistencyNode(
     character_profile=character_profile,
     target_type="image",
     scene_description="在咖啡厅阅读"
   )
   → enhanced_prompt (应用于图像生成)
```
