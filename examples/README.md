# 示例代码

本目录包含ComfyUI-UnlimitAI的使用示例。

---

## 📋 示例列表

### 基础示例

1. **basic_usage.py** - 基础使用示例
   - 文本生成
   - 图像生成
   - 视频生成
   - 音频生成

2. **character_management.py** - 角色管理示例
   - 加载角色图像
   - 定义角色音色
   - 角色一致性维护

3. **workflow_examples.py** - 工作流示例
   - 小说到漫剧转换
   - 批量处理
   - 自定义工作流

### 高级示例

4. **advanced_retry.py** - 高级重试机制
   - 指数退避
   - 自定义重试逻辑
   - 错误处理

5. **cost_optimization.py** - 成本优化示例
   - 模型选择
   - 成本计算
   - 优化策略

6. **parallel_processing.py** - 并行处理示例
   - 异步API调用
   - 批量生成
   - 性能优化

---

## 🚀 快速开始

运行示例：

```bash
# 设置API密钥
export UNITED_API_KEY="your_api_key"

# 运行基础示例
python examples/basic_usage.py

# 运行角色管理示例
python examples/character_management.py
```
